pgsql当前线程数与最大线程数

PostgreSQL 最大连接数：通过 postgresql.conf 文件或 SQL 查询 SHOW max_connections; 获取。
当前连接数：通过 SQL 查询 SELECT count(*), sum(case when state = 'active' then 1 else 0 end) FROM pg_stat_activity; 获取。
