# interpreter

[官方文档](https://docs.openinterpreter.com/getting-started/introduction)

## 安装

```bash
pip install open-interpreter
```

## 使用

```bash
interpreter
# 使用本地模型
interpreter --model ollama/<model-name>
# 示例
interpreter --model ollama/qwen2.5-coder
```
