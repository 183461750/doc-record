# nacos

## 快速开始

打开命令窗口执行：

* Clone project

  ```powershell
  git clone --depth 1 https://github.com/nacos-group/nacos-docker.git
  cd nacos-docker
  ```

* Standalone Derby

  ```powershell
  docker-compose -f example/standalone-derby.yaml up
  ```
